Manipulating CSS Class Properties Using JavaScript

Change Elements by Id

This button toggles the background color of this header to pink and back to default by selecting it from all the other elements via Id.

Change Elements by Class

This button toggles the font style of all elements labeled "by-class" between normal and italic via class. The paragraph after the "CSS Selector" header is not in this class.

Change Elements by Tag

This button aligns the font of all p elements via their HTML Tags.

Change Elements by CSS Selector

This button toggles the font family of all h3 elements between Rock Salt and Antic Slab via CSS Selectors.
