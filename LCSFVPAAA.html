Associative Arrays

Associative Arrays are lists of name/ value pairs where the developer names the keys for each field. They can be referenced by the name of the key.

This is an array about me.

var myself = {name: "Michael", job: "Game Designer", food: "Ice Cream", age: 35};

This example displays my name using the name of the key, instead of the number place.

document.getElementById('spnAArrayKey').innerHTML = myself['name'];

This example displays my name again, but references it like an object property.

document.getElementById('spnAArrayProperty').innerHTML = myself.name;

We can show other fields too. This is the food field.

This example displays tries to display the name again, but cannot because we labeled the keys with names instead of numbers.

document.getElementById('spnAArrayNumber').innerHTML = myself[0];
