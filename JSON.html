JSON Parse and Stringify

JSON Parse

JSON.parse turns a string into a JSON Object (associative array) to be used

JSON.parse Syntax

//Set up the string data
var strGame = '{"name":"Red 7", "description":"A strategic hand management game.", "player_count":"2-4", "age":"10+", "score":4, "difficulty":3}';
//Convert the string to a JSON object
var games = JSON.parse(strGame);
